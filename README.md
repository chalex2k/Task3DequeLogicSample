# Пример реализации логики в 3 таске
1 метод с логикой обрабатывает и стандартные коллекции и самописные.  
- TaskLogic.java содержит метод solve в котором будет реализация логики вашего варианта.  
- SimpleLinkedList.java это реализованный на лекции список. По сути это е есть самописная реализация стека и очереди.  
- SimpleDeque.java класс-обёртка, чтобы наша реализация стека и очереди реализовывала стандартный интерфейс.
